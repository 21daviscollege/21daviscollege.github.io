Circuit Diagram

The wiring setup consists of connecting the potentiometer, an input device, to the analog input pin (A0) of the Arduino. The output of the potentiometer is read by the Arduino to adjust the brightness of the LED connected to PWM-capable pin 3. The LED's brightness is controlled using Pulse Width Modulation (PWM), which changes the amount of time the LED is on during each cycle. The potentiometer is wired such that its middle pin is connected to the Arduino's A0 pin, one of the side pins is connected to the 5V rail, and the other side pin is connected to the GND rail. The LED is connected in series with a suitable current-limiting resistor to pin 9, ensuring proper brightness control.

How PWM Works

Pulse Width Modulation (PWM) is a technique used to simulate analog voltage using digital means. By rapidly switching the LED on and off at varying intervals, we control its brightness. A higher duty cycle (on-time) means a brighter LED.

The Arduino Code


const int analogInPin = A0;  // Analog input pin for the potentiometer
const int analogOutPin = 3;  // PWM output pin for the LED

void setup() {
  Serial.begin(9600);
  Serial.println("Testing analogWrite Function for PWM");
}

void loop() {
  int sensorValue = analogRead(analogInPin);  // Read potentiometer
  int outputValue = map(sensorValue, 0, 1023, 0, 255);  // Map to PWM range

  analogWrite(analogOutPin, outputValue);  // Set LED brightness

  Serial.print("Sensor = ");
  Serial.print(sensorValue);
  Serial.print("\t Output = ");
  Serial.println(outputValue);

  delay(20);  // Short delay for smoother output
}
